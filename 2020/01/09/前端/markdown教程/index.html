标题

  • 使用#来表示标题
# H1
## H2
### H3
#### H4
##### H5
###### H6
  • 使用==和–来表示一级和二级标题 
H1
==
H2
--

段落格式

段落

  • 可以用两个以上空格和一个回车表示换行。
  • 也可以用一个空行来表示重新开始一个段落。

字体

*斜体文字*
_斜体文字_
**加粗文字**
__加粗文字__
***斜体加粗***
___斜体加粗___

斜体文字
斜体文字
加粗文字
加粗文字
斜体加粗
斜体加粗

分割线

  • 三个以上的*，-，_都可以表示分割线。
******************
* * * * * * * * * 
-------------------
____________________

删除线

~~delete line~~

delete line

下划线

  • 下划线可以通过html的<u>标签来实现:
带下划线文本

带下划线文本

脚注

  • 脚注是对文本的补充说明。
[^标注文本]
[^标注文本]: 标注文本

列表

  • 可以使用*,-,+来做为无序列表的标记。
  • 可以使用数字作为有序列表的标记。
  • 列表的嵌套需要加一个tab键(4个空格)。
1. xxxxx
    * xxxxxx
    - xxxxxx
    + xxxxxx
2. xxxxxxx
    1. xxxxxx
    2. xxxxxx
  1. xxxxx
    • xxxxxx
    • xxxxxx
    • xxxxxx
  2. xxxxxxx
    1. xxxxxx
    2. xxxxxx

区块

  • 区块使用>符号加一个空格符号表表示。
  • 区块可以嵌套使用。
  • 如果在列表中使用区块，前面加上4个空格。
> xxxxxxx
> > xxxxxx
> > > xxxxxxx

- xxxx
    > xxxxxx

xxxxxxx

xxxxxx

xxxxxxx

  • xxxx

    xxxxxx

代码

  • 如果是段落上的片段代码可以用code来包含。
  • 代码区域使用4个空格来表示
  • 也可以使用```包裹一段代码，可以指定，也可以不指定。

链接

  • 可以使用的形式表示链接
[这是一个链接](http://gaobo1112.github.io)

这是一个链接

  • 或者也可以直接使用链接。

http://gaobo1112.github.io

  • 链接也可以用变量代替，然后在文章末尾添加变量即可。
[google][1]
[baidu][2]

[1]: www.google.com
[2]: www.baidu.com

google
baidu

插入图片

  • 语法格式如下:
![alt 属性文本](图片地址)
![alt 属性文本](图片地址 "可选标题")
  • 也可以使用变量
  • 但是没发设置图片属性，如果设置，使用html。

表格

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |
  • 对齐方式:
-: 右对齐
:- 左对齐
:-: 居中对齐

高级技巧

  • 使用markdown可以支持html元素，可以插入数学公式，还可以画流程图等。
